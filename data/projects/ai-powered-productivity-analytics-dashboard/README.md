# AI-powered Productivity Analytics Dashboard

## Overview
This project aims to visualize productivity patterns and suggest improvements based on user data using AI techniques.

## Getting Started
1. Clone the repository.
2. Install dependencies for both the server and client.
3. Start the server and client to view the dashboard.

## Technologies Used
- Node.js
- Express
- React
- MongoDB
